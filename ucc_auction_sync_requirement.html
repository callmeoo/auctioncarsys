UCC海外站 · 国内竞价大厅车源同步需求说明

目标：将国内竞价大厅车源同步到海外站，经海外运营审核后展示，用于获取询价 / 找车 / 代采线索。

1. 背景说明

本期做什么国内竞价车辆同步至海外站；海外运营审核上架；海外用户提交询价 / 找车 / 代采意向；国内状态变化后海外自动更新。

核心原因

  • 国内竞价周期短，海外客户决策、销售跟进、保证金确认通常来不及匹配单台车。
  • 海外站前台应弱化“立即竞拍这台车”，强化“Find Similar Cars / Ask UCC to Source”。
  • 线索生命周期必须独立于车辆：车辆结束、成交、流拍后，线索仍可继续跟进。

3. 通用技术规则

规则要求
接口鉴权接口需做鉴权校验，具体鉴权方式由双方技术约定，例如 Token、签名、白名单等。
接口方向数据由国内系统提供给 UCC。若采用国内主动推送，则字段为请求字段；若采用 UCC 拉取，则字段为返回字段。
唯一键domesticVehicleId + auctionId。同一车辆重新上拍视为新记录。
价格同步当前价不逐笔同步，按固定时间间隔同步最新价格，或详情页按需拉取。

提供给 UCC 接口一：车辆列表接口

范围：竞价大厅的在拍车源，含报废车，不含有保留价拍卖、政企专场、顺序拍。

用途：提供海外站列表页所需的车辆基础字段。列表接口只提供列表展示必要信息，详情信息通过车辆详情接口获取。

PUSH / GET 车辆列表接口
/api/auction/vehicles
支持国内按批次主动推送给 UCC；也支持 UCC 按分页、更新时间、竞价状态主动拉取车辆列表。

请求方式说明

场景说明
国内主动推送国内按批次把车辆列表数据推送给 UCC。该方式下字段为请求字段 / 推送字段。
UCC 主动拉取UCC 按分页、更新时间、竞价状态等条件查询国内车辆列表。该方式下字段为返回字段。

UCC 主动拉取查询条件

字段字段要求
page页码，从 1 开始。
pageSize每页数量，建议由双方技术约定最大值。
updatedAfter可选，按数据更新时间增量拉取，格式：yyyy-MM-dd HH:mm:ss。
auctionStatus可选，竞价状态：预展 / 拍卖中 / 成交 / 流拍 / 取消拍卖 / 撮合中。

字段说明

字段字段要求
domesticVehicleId国内车辆 ID，必传。
auctionId竞价 ID，必传；同一车辆重新上拍视为新场次。
vinVIN 码 / 车架号，建议必传，便于车辆唯一识别与海外展示。
brandId品牌 ID，必传。
seriesId车系 ID，必传。
modelId车型 ID，必传。
modelYear年款，例如 2021。
coverImageUrl车辆主图 URL。
cityName车辆所在地城市。
registerDate上牌时间，格式：yyyy-MM 或 yyyy-MM-dd。
mileage表显里程，单位：km。
energyType能源类型，需提供枚举值，便于海外站存储和英文翻译。建议枚举：gasoline 汽油 / diesel 柴油 / hybrid 混动 / ev 纯电 / other 其他。
biddingPrice起拍价，单位：CNY 元。
vehicleGrade车辆评分定级，例如 85A。
auctionEndTime竞价结束时间，格式：yyyy-MM-dd HH:mm:ss。
domesticAuctionStatus国内竞价状态：预展 / 拍卖中 / 成交 / 流拍 / 取消拍卖 / 撮合中。
updatedTime数据更新时间，格式：yyyy-MM-dd HH:mm:ss。

提供给 UCC 接口二：车辆详情接口

范围：竞价大厅的在拍车源，含报废车，不含有保留价拍卖、政企专场、顺序拍。

用途：提供车辆详情页所需的完整车辆信息。详情接口需包含车辆列表接口的全部字段。

PUSH / GET 车辆详情接口
/api/auction/vehicles/{domesticVehicleId}/{auctionId}
可由国内推送详情数据给 UCC，也可由 UCC 根据国内车辆 ID + 竞价 ID 查询车辆详情。

提供给 UCC 接口三：车辆状态通知

范围：竞价大厅的在拍车源，含报废车，不含有保留价拍卖、政企专场、顺序拍。

用途：同步国内车辆竞价状态变化，海外站根据状态更新前台展示和线索标记。

PUSH / GET 车辆状态通知
/api/auction/vehicles/status
可采用国内主动通知 UCC，或 UCC 定时拉取状态变更列表。二选一即可。

状态字段

字段字段要求
domesticVehicleId国内车辆 ID，必传。
auctionId竞价 ID，必传。
status国内竞价状态：预展 / 拍卖中 / 成交 / 流拍 / 取消拍卖 / 撮合中。
statusTime状态发生时间，格式：yyyy-MM-dd HH:mm:ss。
currentPrice可选，状态变化时的当前价，单位：CNY 元。
remark可选，状态说明。

详情页提示

The displayed auction price is for reference only. Final cost may include service fee, export handling fee, logistics, customs and other charges.
新能源 / 带电池车辆需提示：目的国进口与电池运输可能有限制，请先咨询。

原则：线索是独立对象，不依赖车辆是否仍在竞拍。
字段说明

线索状态联动

  • 来源车 sold：线索不关闭，可转同类找车。
  • 来源车 unsold：线索转同类找车需求；如为撮合失败 / 超时，可作为重点线索继续跟进。
  • 来源车 cancelled：标记车辆已取消，销售继续确认客户真实需求。
